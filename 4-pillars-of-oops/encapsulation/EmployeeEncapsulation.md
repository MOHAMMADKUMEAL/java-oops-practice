# Employee Information Management using Encapsulation

## Problem Statement
Create a class `Employee` with private attributes:
- empId
- empName
- designation
- email
- phone
- salary

Use getter and setter methods to access and modify these attributes.

## Input Format
1. empId (int)
2. empName (String)
3. designation (String)
4. email (String)
5. phone (long)
6. salary (double)

## Output Format
Display:
empId  
empName  
designation  
email  
phone  
salary  

## Concept Used
Encapsulation (private variables + public getter/setter methods)
