📚 Library Book Management System (Java)

This project is a simple Library Book Management System built using Java. It demonstrates core Object-Oriented Programming (OOP) concepts like encapsulation using a Book class with private attributes and getter/setter methods.

🚀 Features
Store book details: ISBN, Title, Author, Price
Encapsulation using private variables
Getter and Setter methods
User input using Scanner
Display book details
🧠 Concepts Used
Object-Oriented Programming (OOP)
Encapsulation
Classes and Objects
Getter and Setter Methods
Java Input/Output

▶️ How to Run
Compile the program:
javac LibraryBookManagement.java
Run the program:
java LibraryBookManagement

The program takes four inputs:

Author (String)
Price (float)
